# Guidelines

Guidelines are shared rules for GrapheneOS app development. They should be
written as durable guidance that applies across apps, not as notes about a
single repository.

This section includes architecture, high-level development principles, and
concrete implementation guidance for Kotlin, Compose, coroutines, dependency
injection, code style, and unit testing.
